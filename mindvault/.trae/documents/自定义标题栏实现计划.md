
# 自定义标题栏实现计划

## 1. 需求分析
用户希望将 Electron 系统的默认标题栏（黑乎乎的那一道）替换为自定义标题栏，以获得更统一和美观的视觉效果。

## 2. 实现方案概述
- **步骤1**：修改主进程 `src/main/index.ts`，设置 `frame: false` 和 `titleBarStyle: 'hidden'` 来隐藏系统标题栏
- **步骤2**：在 `Header.tsx` 顶部添加自定义标题栏组件
- **步骤3**：实现自定义标题栏的拖拽功能
- **步骤4**：添加窗口控制按钮（最小化、最大化/还原、关闭）
- **步骤5**：调整现有 Header 高度，适应新的标题栏

## 3. 文件修改清单
- `d:\Android\Code\naodong\mindvault\src\main\index.ts` - 配置无框架窗口
- `d:\Android\Code\naodong\mindvault\src\renderer\components\layout\Header.tsx` - 添加自定义标题栏组件
- `d:\Android\Code\naodong\mindvault\src\renderer\utils\api.ts` - 添加窗口控制IPC接口（可能需要）
- `d:\Android\Code\naodong\mindvault\src\main\ipc\index.ts` - 注册窗口控制处理函数（可能需要）

## 4. 详细实现步骤

### 4.1 配置无框架窗口
在 `src/main/index.ts` 的 `createWindow` 函数中，修改 `BrowserWindow` 配置：
- 添加 `frame: false`
- 添加 `titleBarStyle: 'hidden'`
- 保留原有的窗口尺寸等配置

### 4.2 添加自定义标题栏到 Header 组件
在 `Header.tsx` 的最外层添加自定义标题栏区域，包含：
- 左侧：窗口标题"脑洞集"
- 中间：可拖拽区域（全屏拖拽）
- 右侧：窗口控制按钮（最小化、最大化/还原、关闭）

### 4.3 实现窗口控制功能
通过 Electron API 或预定义的 IPC 消息来实现：
- 最小化
- 最大化/还原
- 关闭窗口
- 拖拽移动窗口

### 4.4 调整现有布局
- 增加 `--header-height` 变量值，或在现有 Header 顶部叠加新的标题栏
- 确保标题栏使用 `var(--header-bg)` 保持与主题一致

## 5. 潜在依赖和注意事项
- 需要确保 IPC 通信正常工作（窗口控制）
- 自定义标题栏需要应用 `app-region: drag` 和 `app-region: no-drag` CSS 属性
- 窗口控制按钮需要添加适当的悬停效果和视觉反馈
- 自定义标题栏高度建议设置为 32px - 36px 之间

## 6. 风险处理
- 如果 IPC 通信有问题，使用 Electron 预加载脚本 (preload.js) 暴露 API
- 如果拖拽功能不流畅，调整 `app-region: drag` 区域
- 保留 `Menu.setApplicationMenu` 以支持快捷键功能
