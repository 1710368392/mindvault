# 为什么视频跟标题不是一个层级？

## 回答

视频和标题**确实是同一个层级**的！它们都是 `.flashback-card` 的直接子元素。

看代码结构（第960-1060行）：

```
.flashback-card  ← 卡片容器
  ├─ div（类型标签行：缩略图 + "视频" + 时间）  ← 第961行
  ├─ h4（标题）                                  ← 第977行
  └─ FlashbackVideoPlayer（视频）                ← 第1036行
```

标题和视频是平级的兄弟元素，都在 `.flashback-card` 里面。

## 那为什么视觉上"留白"只在视频两侧出现？

因为：
- **标题**是文字，作为块级元素默认 `width: 100%` 撑满卡片宽度 → 没有留白
- **视频**设了 `width: fit-content` + `margin: 0 auto`，宽度收缩到视频自身宽度（如253px）→ 两侧出现留白

标题撑满了，视频没撑满，所以只有视频两侧有留白。

## 解决方案

既然视频和标题是同级的，最简单的办法就是：**让视频也撑满卡片宽度**，跟标题一样。

具体做法：
1. FlashbackVideoPlayer 容器：`width: 100%`（撑满卡片，跟标题一样）
2. video 元素：`width: 100%` + `object-fit: contain`（保持比例，不裁剪）
3. 容器背景色：`rgba(0,0,0,0.03)`（与 .flashback-card 一致，contain 产生的上下留白与卡片背景融为一体）
4. 去掉 `margin: 0 auto` 和 `fit-content`
5. 去掉竖屏视频的 scaledWidth 计算逻辑（不再需要，contain 会自动处理）

这样视频就跟标题一样撑满卡片宽度，竖屏视频上下会有少量与卡片背景同色的留白，但不会有两侧的突兀留白。
