## 问题详情

### 🔴 严重问题

#### BUG-023：CanvasView中longPressTimerRef未声明导致运行时错误

- **严重程度**：🔴 严重
- **优先级**：高
- **发现位置**：CanvasView.tsx:838
- **问题描述**：使用了`longPressTimerRef`变量，但该变量从未声明，在代码中有19处使用
- **影响**：画布页核心功能受损，可能导致应用崩溃
- **修复建议**：在组件顶部添加`const longPressTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null);`

---

### 🟠 一般问题

#### BUG-024：handleConnectorClick调用addCanvasEdge参数不匹配

- **严重程度**：🟠 一般
- **优先级**：高
- **发现位置**：CanvasView.tsx:1276-1288
- **问题描述**：调用`addCanvasEdge`时只传了4个参数，但函数已更新为6个参数（新增了sourceConnector和targetConnector）
- **影响**：连线创建功能可能异常

#### BUG-025：CanvasView中缺少UI工具栏组件的引用

- **严重程度**：🟠 一般
- **优先级**：中
- **发现位置**：CanvasView.tsx:1523
- **问题描述**：提到了`canvas-toolbar`类，但缺少完整的工具栏UI组件（视图切换按钮、缩放控制、添加创意按钮等）
- **影响**：用户无法直观切换工具模式

#### BUG-026：CanvasView中画布节点卡片组件未完全实现

- **严重程度**：🟠 一般
- **优先级**：中
- **问题描述**：引用了`canvas-node-card`类，但缺少完整的节点卡片UI实现（卡片样式、连接点显示、拖拽逻辑等）
- **影响**：卡片可能无法正常显示和交互

#### BUG-027：CanvasView中连线渲染组件缺失或不完整

- **严重程度**：🟠 一般
- **优先级**：中
- **问题描述**：需要渲染贝塞尔曲线连线、临时连线、连接点编辑等，但可能缺少完整的SVG/Canvas渲染组件
- **影响**：连线可能不显示或交互功能缺失
