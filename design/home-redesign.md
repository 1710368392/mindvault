## 三、设计规范

### 欢迎条
- 高度：约80px（padding: 20px 28px）
- 背景：`linear-gradient(135deg, var(--primary-color), var(--primary-light))`
- 文字：白色
- 统计数字：22px 加粗，标签 11px 半透明
- 装饰：2个半透明圆形（右上角）
- "开始记录"按钮：半透明白色背景 + 毛玻璃效果

### 快捷操作
- 布局：5列等宽网格
- 间距：12px
- 每个按钮：图标(40×40) + 标题(13px) + 描述(11px)
- 图标背景：对应颜色的10%透明度
- Hover：上移2px + 阴影加深

### 灵感闪回
- "换一个"按钮：主色边框 + 主色背景浅色
- 刷新图标：加载时旋转动画
- 卡片：点击可预览详情

### 最近创意
- 列表项：图标(32×32) + 标题 + 时间 + 优先级星标
- 最多显示6条
- 支持右键菜单、拖拽
- Hover：背景变色

### 最近看板
- 最多显示3个
- 每个看板：图标 + 名称 + 时间 + 箭头
- Hover：背景变主色浅色 + 右移2px

## 五、动效设计

| 元素 | 动效 | 参数 |
|------|------|------|
| 欢迎条 | 淡入+上移 | opacity 0→1, y 20→0, 0.4s |
| 快捷操作 | 依次淡入+上移 | 每项延迟 0.05s |
| 灵感闪回 | 缩放+淡入 | scale 0.95→1, 0.25s |
| 最近创意列表项 | 依次淡入+上移 | 每项延迟 0.04s |
| 最近看板 | 依次淡入+上移 | 每项延迟 0.05s |
| 快捷操作Hover | 上移+阴影 | translateY(-2px), 0.15s |
| 看板Hover | 右移+背景变色 | translateX(2px), 0.15s |

## 六、优化前后对比

### 优化前问题
1. 欢迎横幅占约30%垂直空间，仅一个"开始记录"按钮
2. 4个统计卡片2×2排列，每卡仅一个数字
3. 无快捷操作入口
4. 无最近看板入口
5. 灵感闪回刷新按钮小、不明显

### 优化后改进
1. 欢迎条紧凑（~80px），统计数字内嵌右侧
2. 统计从4个大卡片→4个数字横排，节省约60%空间
3. 新增5个快捷操作按钮（快速录入/搜索/模板库/统计/快捷键）
4. 新增最近看板区域（最多3个）
5. 灵感闪回"换一个"按钮更突出
6. 最近创意从5条→6条
7. 整体信息密度提升约40%
