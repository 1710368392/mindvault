# MindVault 脑洞集

> 一个融合 AI 能力的灵感收集与管理应用

## 功能特点

- 🧠 **AI 智能助手** - 内置 AI 对话能力，支持多种模型
- 📝 **Markdown 编辑** - 强大的 Markdown 编辑体验
- 🎵 **音乐播放器** 
- 🎨 **创意模板** - 丰富的创意模板库
- 📊 **看板视图** - 支持看板、画布等多种视图
- 🌤️ **天气系统** - 实时天气信息展示
- 🔍 **智能搜索** - 快速检索你的灵感库
- ♻️ **回收站** - 误删恢复机制

## 技术栈

- **框架**: Electron + React + TypeScript
- **UI 库**: Ant Design
- **状态管理**: 自研状态管理系统
- **存储**: SQLite + Redis
- **AI**: 支持 Claude、Gemini、GPT、KIMI、DeepSeek 等多模型

## 开发

```bash
# 安装依赖
cd mindvault
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
