# 脑洞集 (MindVault) — 产品需求文档

> 本文档由产品经理基于代码审查和项目总看板生成，作为后续开发的唯一需求依据。

---

## 一、应用概述

### 1.1 产品定位

**脑洞集（MindVault）** 是一款面向创意工作者的桌面灵感记录与整理工具。核心价值是帮助用户随时捕捉灵感、整理思绪、可视化创意关联。

### 1.2 目标用户

- 写作者（网文、剧本、散文等）
- 产品经理 / 设计师（灵感收集、方案构思）
- 学习者（笔记整理、知识关联）
- 任何需要记录和整理碎片化想法的人

### 1.3 技术栈

| 类别 | 技术 |
|------|------|
| 桌面框架 | Electron 33+ |
| 前端框架 | React 18 + TypeScript 5 |
| 状态管理 | Zustand 4 |
| 构建工具 | Vite 6 |
| 样式方案 | Tailwind CSS 4 + CSS 变量 |
| 动画库 | Framer Motion 11 |
| 数据可视化 | ECharts 5 |
| 数据库 | Better-SQLite3 |
| 路由 | React Router DOM 6 |

### 1.4 当前版本

v1.0.0 — 已完成基础框架搭建和核心功能实现，但存在 **63 个已知问题**（详见项目总看板），其中包括 **10 个阻断性 Critical Bug**，严重影响核心功能使用。

### 🥇 Top 1：集成 Markdown 编辑器

**RICE 评分**：高

| 维度 | 评分 | 理由 |
|------|------|------|
| Reach | 10/10 | 所有用户都需要编辑创意内容 |
