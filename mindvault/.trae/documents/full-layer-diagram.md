## 宽度传递链（竖屏视频 540×960）

```
home-grid 第1列（1fr ≈ 400px）
  └→ Card body（撑满列宽 ≈ 400px）
      └→ Carousel（撑满 Card body ≈ 400px）
          └→ Carousel 内部 slide（100% ≈ 400px）
              └→ div key={fb.id}（fit-content → 跟着子元素走）
                  └→ .flashback-card（fit-content → 跟着子元素走）
                      └→ FlashbackVideoPlayer（width: 253px，被 playerStyle 覆盖）
                          └→ video（width: 100% = 253px）
```

**关键**：从 div key={fb.id} 开始，每一层都是 `fit-content`，宽度由最内层的视频决定，逐层向上传递。视频 253px → FlashbackVideoPlayer 253px → .flashback-card 269px（253+8+8 padding）→ div key 269px → Carousel slide 400px（不变，但内容居中）。
